Sobre el curso

Este curso completo de Java te llevará desde los fundamentos de programación hasta el desarrollo de aplicaciones empresariales. Aprenderás sobre sintaxis, estructuras de control, POO, manejo de excepciones, colecciones, bases de datos con JDBC y frameworks modernos como Spring Boot. Con más de 50 horas de contenido y prácticas guiadas, estarás listo para crear software robusto, eficiente y escalable. Al finalizar, obtendrás una certificación oficial que avala tu dominio de Java, uno de los lenguajes más demandados en el mercado laboral global.

Módulos del curso

Módulo 1

Introducción a Java

Conoce el entorno de desarrollo y fundamentos del lenguaje

  • Historia y características de Java
  • Instalación y configuración de IntelliJ IDEA
  • Estructura de un programa en Java
  • Compilación y ejecución
Módulo 2

Sintaxis y Variables

Aprende los fundamentos de la sintaxis Java

  • Tipos de datos primitivos
  • Declaración de variables
  • Operadores aritméticos y lógicos
  • Entrada y salida con Scanner
Módulo 3

Control de Flujo

Estructuras condicionales y bucles

  • if, else, switch
  • for, while, do-while
  • Break y continue
  • Ejercicios prácticos
Módulo 4

Programación Orientada a Objetos

Aprende el corazón de Java: la POO

  • Clases y objetos
  • Encapsulación y getters/setters
  • Herencia y polimorfismo
  • Constructores y métodos
Módulo 5

Colecciones y Arrays

Estructuras dinámicas de almacenamiento

  • Arrays y matrices
  • ArrayList y LinkedList
  • HashMap y HashSet
  • Iteradores y streams
Módulo 6

Manejo de Excepciones

Control de errores en tiempo de ejecución

  • Try-catch-finally
  • Excepciones personalizadas
  • Lanzamiento con throw y throws
  • Buenas prácticas
Módulo 7

Archivos y Entrada/Salida

Lectura y escritura de datos

  • Clases File y FileReader
  • BufferedReader y BufferedWriter
  • Serialización de objetos
  • Manejo de excepciones de IO
Módulo 8

Interfaces y Clases Abstractas

Diseño flexible y extensible

  • Interfaces funcionales
  • Clases abstractas
  • Implementaciones múltiples
  • Ejemplo práctico
Módulo 9

Bases de Datos con JDBC

Conecta Java con bases de datos SQL

  • Conexión con MySQL
  • Consultas CRUD
  • PreparedStatement
  • Manejo de transacciones
Módulo 10

Desarrollo Web con Spring Boot

Crea APIs y aplicaciones empresariales

  • Arquitectura MVC
  • Controladores y servicios
  • Endpoints REST
  • Integración con bases de datos
Módulo 11

Testing y Buenas Prácticas

Mejora la calidad de tu código

  • JUnit y Mockito
  • Test-Driven Development
  • Principios SOLID
  • Refactorización
Módulo 12

Proyecto Final

Desarrolla una aplicación Java completa

  • Diseño del proyecto
  • Desarrollo con POO y JDBC
  • Implementación web con Spring Boot
  • Presentación y certificación
